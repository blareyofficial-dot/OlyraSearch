# 🌌 Olyra Search  
A fast, lightweight, open‑source Python web browser created by **Olyra**, a personal digital company.  
Built with PyQt5 and QtWebEngine, Olyra Search features a modern purple‑and‑blue UI, animated gradient buttons, tabbed browsing, encrypted password storage, and a clean “New Tab” experience — all inside a **single Python file**.

---

## ✨ Overview  
**Olyra Search** is designed to be simple, elegant, and customizable.  
It’s built entirely in Python, making it easy to modify, extend, and compile into a Windows executable.  
The project is fully open‑source under the **MIT License**, allowing community contributions and enabling free code‑signing certificates to avoid Windows Smart App Control warnings.

---

## 🚀 Features  
- **Tabbed browsing**  
- **Custom New Tab page** (`https://olyrasearch.base44.app`)  
- **“New tab” label instead of showing the URL**  
- **Back & Reload buttons**  
- **Soft gradient‑animated buttons**  
- **Purple/blue modern UI theme**  
- **Bookmarks system**  
- **History tracking**  
- **Clear history button**  
- **Encrypted password manager (Fernet)**  
- **Single‑file architecture**  
- **Easy `.exe` conversion via pytoexe.top**  
- **Open‑source friendly for code‑signing certificates**

---

## 🛠️ Tech Stack  
- Python 3.x  
- PyQt5  
- PyQtWebEngine  
- cryptography (Fernet)  
- Qt Stylesheets  
- QPropertyAnimation
